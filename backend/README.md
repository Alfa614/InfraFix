# InfraFix Backend (Express + Prisma + SQLite)

## Quick start

```bash
npm install
npm run prisma:generate
npm run prisma:migrate
npm run seed
npm run dev
```

API at `http://localhost:4000`

Default seed users:
- Admin: `admin@infrafix.local` / `admin123`
- User: `user@infrafix.local` / `user123`
